# Webutvikling prosjekt 2
Prosjektet skal visualisere GitLab data. For å gjøre dette benytter vi oss av GitLab sitt eget api.

Kommandoer:
### `npm start`
For å starte prosjektet
### `npm test`
For å teste prosjektet
### `npm run lint`
For å linte prosjektet
